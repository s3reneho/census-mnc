# Census MNC paper 
This repository contains the R and Python processing pipeline for analyzing MNC-related import transactions.

## Project Structure

- `python/` – Python script for extracting data
- `src/` – R scripts for cleaning and matching
- `data/` – Small reference files only (no raw extracts)
- `outputs/` – Processed outputs and summary tables
- `docs/` – Run logs and documentation
- `notebooks/` – Exploratory analysis

## Workflow

1. Run Python scraping code
2. Download output locally
3. Perform matching and cleaning in R

## Data Policy

Raw administrative or proprietary data should NOT be committed to this repository.
Only code, documentation, and non-sensitive outputs should be stored here.
